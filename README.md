# Followers CMS — панель управления продуктами

Веб-админка, которая заменяет управление цифровыми продуктами через Google Sheets.
Каждый продукт = отдельный лист исходной таблицы; внутри — список **вариантов онбординга**
с тарифами, инапной сеткой (паками) и апсейлами.

Это **фронт-прототип**: данные хранятся в `localStorage` браузера и засеяны из таблицы
`Followers_Products.xlsx` (файл `seed-data.js`). Бэкенда пока нет — но структура данных уже
готова, чтобы в любой момент завести API/БД и синхронизировать вместо таблицы.

## Запуск

```bash
cd followers-cms
python3 -m http.server 8000
# открыть http://localhost:8000
```

Либо просто открыть `index.html` в браузере.

## Доступ (демо)

```
логин:  admin
пароль: followers2026
```

Креды зашиты в `app.js` (`CREDS`) — это заглушка-гейт, а не настоящая авторизация.
При подключении бэкенда логин нужно перенести на сервер.

## Что умеет

- **4 продукта** в сайдбаре: UpFollow, Followerse, Wordbeat, Planly (с количеством вариантов).
- **Карточки вариантов** со статусом (Активен / На стопе), variant key, ссылкой и счётчиками
  тарифов/паков/апсейлов.
- **Поиск** по названию / variant key / ссылке и **фильтр** по статусу.
- **Редактор варианта** (боковая панель): статус, название, variant key, ссылка, флоу отмены,
  VAT, локализация + языки, логика начисления, и вложенные редактируемые списки:
  - **Тарифы подписки** — продукт / цена / billing ID (Recurly или Truegate);
  - **Инапная сетка (паки)** — название + product code;
  - **Апсейлы** — основной/скидочный варианты с product code.
- **Создание / удаление** вариантов.
- **Экспорт всей конфигурации в JSON** и **сброс** к исходным данным из таблицы.

## Структура данных

```
products[]
  ├─ id, name, billingLabel
  └─ variants[]
       ├─ status, name, link, variantKey, vat, localizationOn, languages[],
       │  accrualLogic, cancelFlow
       ├─ plans[]   { name, price, billingId }
       ├─ packs[]   { name, code }
       └─ upsells[] { title, main, mainCode, discount, discountCode }
```

## Файлы

| Файл           | Назначение                                            |
|----------------|-------------------------------------------------------|
| `index.html`   | разметка: экран логина, оболочка, редактор-дровер      |

| `seed-data.js` | данные, сгенерированные из `Followers_Products.xlsx`   |
